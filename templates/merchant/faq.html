Bắt đầu với ZaloPay

ZaloPay Merchant là nền tảng thanh toán dành cho doanh nghiệp, cho phép bạn nhận thanh toán từ khách hàng qua nhiều phương thức khác nhau như QR code, thẻ ngân hàng, ví điện tử. Hệ thống cung cấp API dễ tích hợp và dashboard quản lý giao dịch chi tiết.

  1. Truy cập trang đăng ký merchant
  2. Điền thông tin doanh nghiệp và người đại diện
  3. Upload các giấy tờ cần thiết (GPKD, CCCD, ...)
  4. Chờ team xác minh (1-2 ngày làm việc)
  5. Nhận thông báo kích hoạt và bắt đầu sử dụng

  • Doanh nghiệp: Giấy phép kinh doanh, Quyết định thành lập
  • Cá nhân: CCCD/CMND, Giấy phép kinh doanh (nếu có)
  • Ngân hàng: Sao kê tài khoản 3 tháng gần nhất
  • Người đại diện: CCCD/CMND của người đại diện pháp lý

Tích hợp API

Không, API ZaloPay được thiết kế đơn giản và dễ sử dụng. Chúng tôi cung cấp:
  • Tài liệu API chi tiết với examples
  • SDK cho các ngôn ngữ phổ biến (PHP, Java, .NET, Node.js)
  • Sandbox để test miễn phí
  • Support team hỗ trợ 24/7

Có, ZaloPay hỗ trợ webhook để thông báo real-time về trạng thái giao dịch. Bạn có thể cấu hình webhook URL trong dashboard để nhận thông báo tự động khi giao dịch hoàn thành, thất bại hoặc có vấn đề.

Có, chúng tôi cung cấp môi trường Sandbox hoàn toàn miễn phí với:
  • API endpoints riêng cho test
  • Test credentials và test data
  • Simulation các trường hợp thành công/thất bại
  • Dashboard để theo dõi test transactions

Thanh toán

  • Ví ZaloPay: Thanh toán trực tiếp từ ví
  • Thẻ ngân hàng: Visa, MasterCard, JCB (nội địa & quốc tế)
  • ATM: Thẻ ATM nội địa
  • Internet Banking: Hầu hết các ngân hàng lớn
  • QR Code: Quét mã thanh toán
  • Trả góp: Các gói trả góp 0% (theo điều kiện)

  • Ví ZaloPay: Tức thì
  • Thẻ ngân hàng: 1-3 phút
  • Internet Banking: 1-5 phút
  • ATM: 1-5 phút
Trong trường hợp có vấn đề, hệ thống sẽ tự động retry và thông báo kết quả.
